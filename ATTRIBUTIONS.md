# Attributions

This project includes a small set of bundled third-party creative assets and tools. This file exists so the credits are easy to find instead of being buried in a README paragraph.

## Audio

- Desktop sound effects in `desktop/src/assets/*.wav` were created with [Bfxr](https://www.bfxr.net/).

## Fonts

- `Pixelify Sans` in `desktop/src/assets/fonts/pixelify-sans` is a Google Fonts family.
- `Silkscreen` in `desktop/src/assets/fonts/silkscreen` is a Google Fonts family.
- General Google Fonts licensing and repository information: [google/fonts](https://github.com/google/fonts)

## Design references

- Some UI direction came from user-provided reference boards and screenshots during development.
- Those reference materials were used for visual guidance only and are not redistributed in this repository as app assets.
